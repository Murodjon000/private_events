# Rails Private Event!

We  built a site similar to a private Eventbrite which allows users to create events and then manage user signups. Users can create events and send invitations and parties (sound familiar?). - Rails

## Installing

- Clone the repo `https://github.com/Murodjon000/private_events.git`
- `cd` into the directory
- Run `bundle install`
- Run `rails db:create db:migrate` (optional)
- Run `rails s`
- Go to `localhost:3000`
